# chooseArea
## 微信小程序-省市区县三级联动选择器
```2018.10.25 更新```

有人反映弹出选择器控件时，不滑动直接点击“确定”按钮没有值返回显示。其实默认坐标是[0,0,0]的，只是没有更新数据而已。现已更正。

```2017.07.21 17:30 更新```

之前版本是从头写到尾没有封装的，有的同学可能看的一脸懵逼，昨天我用微信官方提供的WXML模板（template）对代码进行了封装，在模板中定义代码片段，通过封装以后我们就可以在不同的地方调用了。
